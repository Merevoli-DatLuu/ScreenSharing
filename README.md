The screen sharing tool.

## Giới thiệu
Đây là công cụ chia sẻ màn hình đơn giản.

## Sử dụng

Khởi động server (phía chia sẻ màn hình)
```
python3 server
```

Khởi động client (phía được chia sẻ)
```
python3 client
# Nhập host và port của server
# Nhấn q để dừng
```
